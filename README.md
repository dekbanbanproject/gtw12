## การใช้ git Install
** ดึงข้อมูลลงมา<br/>
1.git clone https://github.com/dekbanbanproject/gtw12.git
<br/>
2.เปลี่ยนชื่อไฟล์ example.env to .env เพื่อเชื่อมต่อ Database
<br/>
3.ใช้คำสั่ง composer update
<br/>
4.php artisan migrate
5.chmod -R apache:apache wwwDirectory
